# Mensagens Modelo — CS DVE Assessoria

**Para:** Davi (CS) e qualquer CS futuro da DVE
**Canal principal:** WhatsApp (grupo do cliente)
**Tom:** Profissional e amigável. Direto, sem enrolação. Humano, sem ser informal demais.

**Regras de ouro:**
- Nunca usar travessão (—)
- Não usar entusiasmo exagerado ("Incrível!", "Que demais!", "Arrasou!")
- Não começar mensagem com "Oi tudo bem?" seguido imediatamente de um pedido
- Usar o primeiro nome do cliente, nunca "prezado(a)"
- Revisar antes de enviar: a mensagem faz sentido sem contexto anterior?

---

## 1. SOCIAL MEDIA

### 1.1 Envio do Planejamento Editorial Mensal

**Quando enviar:** assim que o planejamento de conteúdo do mês estiver aprovado internamente pela Ariana/Danilo.

**Mensagem padrão:**
```
Oi [nome]! Tudo bem?

O planejamento de conteúdo de [mês] está pronto.

[link do documento]

Passa uma olhada quando puder e me fala se tem algo que você quer ajustar, acrescentar ou tirar antes de começarmos a produção.
```

**Variação quando tem tema de destaque no mês:**
```
Oi [nome]!

O planejamento de [mês] está pronto. Esse mês a gente focou em [tema principal, ex: depoimentos de clientes / lançamento do produto X / datas sazonais do segmento].

[link do documento]

Qualquer ajuste, me fala!
```

**Nota operacional:** não enviar na sexta à tarde nem na segunda de manhã. Melhor entre terça e quinta, até 14h.

---

### 1.2 Envio de Artes Estáticas para Aprovação

**Quando enviar:** quando o Wesley finalizar o lote de artes.

**Mensagem padrão:**
```
Oi [nome]! As artes de [descrição, ex: "anúncios de maio" / "campanha de captação" / "posts da semana"] estão prontas para aprovação.

[link do Drive / pasta]

São [número] peças no total. Me fala o que achou!
```

**Quando há urgência (campanha com data para subir):**
```
Oi [nome]! As artes de [descrição] estão prontas.

[link]

Precisamos da sua aprovação até [data/horário] para conseguir subir os anúncios dentro do prazo. Qualquer ponto para ajustar, já manda aqui.
```

**Variação para cliente que costuma pedir muitas revisões (calibrar expectativa antes):**
```
Oi [nome]! As artes ficaram prontas.

[link]

Passa uma olhada e me fala o que quer ajustar. A gente faz até [X] rodadas de revisão dentro do escopo.
```

---

### 1.3 Envio de Vídeos / Reels para Aprovação

**Quando enviar:** quando Daniela ou Kauan finalizarem a edição.

**Mensagem padrão:**
```
Oi [nome]! O vídeo de [descrição, ex: "apresentação do serviço" / "reel da semana"] está pronto para você ver.

[link]

Me fala o que achou. Se tiver algum ajuste, já descreve aqui que a gente passa para edição.
```

**Quando são vários vídeos de uma vez:**
```

[link da pasta]

São [X] vídeos ao todo. Passa uma olhada e me dá o feedback de cada um, pode ser aqui mesmo no chat ou direto no link com comentários.
```

---

### 1.4 Confirmação de Publicação

**Quando enviar:** após publicar um conteúdo importante (campanha nova, post de data especial, lançamento).

**Mensagem padrão:**
```
Oi [nome]! [Anúncio / Post / Reel] de [descrição] já está no ar.

[link da publicação, se aplicável]

Qualquer coisa que queira ajustar ainda, me fala.
```

## 2. TRÁFEGO PAGO

### 2.1 Envio do Relatório Mensal de Tráfego

**Quando enviar:** entre os dias 1 e 5 do mês seguinte, após o relatório ser gerado.

**Mensagem padrão:**
```
Oi [nome]! O relatório de [mês] está pronto.

[link do relatório]

Resumo rápido: [1 linha com o principal destaque do mês, ex: "custo por lead caiu 18% em relação ao mês anterior" / "campanhas de Google performaram acima da meta" / "Meta ficou estável, Google trouxe os melhores resultados"].

Quando quiser conversar sobre os números, é só falar que a gente marca.
```

**Variação quando o mês foi fraco (não esconder, ser direto):**
```
Oi [nome]! O relatório de [mês] está pronto.

[link do relatório]

Esse mês tivemos um resultado abaixo do esperado em [métrica]. Já estamos com os ajustes em andamento para [próximo mês] — detalhei isso no relatório. Se quiser conversar sobre isso antes da nossa próxima reunião, pode chamar.
```

**Variação quando o mês foi acima da meta:**
```
Oi [nome]! O relatório de [mês] está pronto.

[link do relatório]

Bom mês: [métrica principal] ficou em [resultado], [comparação positiva]. O relatório traz o detalhamento completo.
```

---

### 2.2 Atualização de Ajuste nas Campanhas

**Quando enviar:** após fazer uma otimização relevante ou mudança de estratégia nas campanhas.

**Mensagem padrão:**
```
Oi [nome]! Passando para te atualizar.

Fizemos um ajuste nas campanhas: [descrição simples do que foi feito, ex: "pausamos os anúncios que estavam com custo alto e aumentamos o orçamento nos que estão performando melhor"].

[Se aplicável: "Vamos acompanhar os resultados nos próximos dias. Qualquer dúvida, me chama."]
```

---

### 2.3 Alerta de Performance (quando algo está fora do normal)

**Quando enviar:** quando uma métrica importante sair muito do padrão, para cima ou para baixo.

**Mensagem padrão:**
```
Oi [nome]! Só um aviso rápido.

Estamos monitorando [métrica] que está [acima/abaixo] do padrão nos últimos dias. [Explicação breve, ex: "Pode ser reflexo do feriado da semana passada" / "Já identificamos o ponto e estamos ajustando"].

[Se precisar de algo do cliente: "Precisamos que você [ação específica, ex: revise o orçamento / confirme se houve alguma mudança no negócio]."]

Qualquer dúvida, estou aqui.
```

## 3. REUNIÕES

### 3.1 Lembrete de Reunião

**Quando enviar:** no dia anterior ou na manhã do dia da reunião.

**Mensagem padrão:**
```
Oi [nome]! Lembrando da nossa reunião [amanhã / hoje] às [horário].

[Link do Google Meet, se aplicável]

Se precisar reagendar, me avisa com antecedência!
```

---

### 3.2 Envio da Minuta Pós-Reunião

**Quando enviar:** em até 1h após o término da reunião.

**Mensagem padrão:**
```
Oi [nome]! Segue o resumo da nossa reunião de hoje.

[link da minuta no ClickUp]

Qualquer ponto que queira corrigir ou acrescentar, me fala.
```

**Nota operacional:** a minuta é gerada pela skill /minuta. O CS só envia o link do ClickUp com a mensagem acima. Não resumir a minuta no WhatsApp, o link já tem tudo.

---

### 3.3 Próximos Passos Pós-Reunião (quando há ação do cliente pendente)

**Quando enviar:** junto com a minuta ou logo após, quando ficou combinado algo que depende do cliente.

**Mensagem padrão:**
```
[Junto da minuta ou logo depois]

De [cliente] ficou pendente:
- [ação 1]
- [ação 2]

Prazo combinado: [data]. Assim que tiver, me manda aqui.
```

## 4. CONTATO PROATIVO

O contato proativo não é só um "oi, tudo bem?". O objetivo é manter presença, mostrar que a DVE está ativa e gerar abertura para o cliente trazer dúvidas, feedbacks ou ideias.

**Clientes Tipo A:** contato todo dia útil
**Clientes Tipo B:** contato 3x por semana (segunda, quarta, sexta funciona bem)

### 4.1 Atualização de Trabalho em Andamento

**Quando usar:** quando tem algo em produção e quer mostrar que o time está trabalhando.

```
Oi [nome]! O time está trabalhando no [entregável, ex: "planejamento de julho" / "artes da próxima campanha"]. Deve ficar pronto até [data]. Qualquer coisa, estou por aqui.
```

---

### 4.2 Compartilhando Dado Relevante

**Quando usar:** quando tem um número bom para mostrar, uma tendência interessante ou algo que o cliente precisa saber.

```
Oi [nome]! Acompanhando as campanhas e vi que [dado relevante, ex: "o custo por lead caiu 20% essa semana" / "o post de [tema] está com engajamento acima da média"].

Bom sinal. Continuamos acompanhando.
```

---

### 4.3 Check-in Simples (quando não tem novidade concreta)

**Quando usar:** para manter presença nos dias que não tem entregável ou dado para compartilhar. Evitar fazer isso todo dia, misturar com outras abordagens.

```
Oi [nome]! Tudo bem?

Alguma dúvida ou algo que queira conversar sobre [área, ex: as campanhas / o conteúdo do mês]?
```

---

### 4.4 Antes de uma Data Importante do Cliente

**Quando usar:** quando o cliente tem um evento, lançamento ou data relevante chegando.

```
Oi [nome]! [Data / evento] está chegando. Estamos com [entregável específico] em andamento para apoiar. Qualquer coisa que precisar antes disso, me fala.
```

## 5. OPERACIONAL / OUTROS

### 5.1 Solicitação de Material ao Cliente

**Quando usar:** quando o time interno precisa de algo que só o cliente pode fornecer (foto, vídeo, informação, acesso).

**Mensagem padrão:**
```
Oi [nome]! Para [finalidade, ex: montar o planejamento de julho / criar os anúncios da campanha], precisamos de:

- [item 1]
- [item 2]
- [item 3]

Consegue mandar até [data]?
```

**Variação quando é urgente:**
```
Oi [nome]! Precisamos de [material específico] hoje para não atrasar [entregável]. Consegue mandar?
```

---

### 5.2 Aviso de Pendência (quando o cliente não respondeu)

**Quando usar:** quando ficou algo pendente do cliente e o prazo está se aproximando ou já passou.

**Mensagem padrão (1º aviso, dentro do prazo):**
```
Oi [nome]! Passando para lembrar que ainda estamos aguardando [item pendente] para conseguir avançar com [entregável / campanha].
```

**Mensagem padrão (2º aviso, prazo passando):**
```
Oi [nome]! Ainda estamos aguardando [item pendente]. Sem isso, não conseguimos [consequência clara, ex: "subir os anúncios" / "publicar o conteúdo da semana"]. Me fala quando der!
```

---

### 5.3 Confirmação de Ajuste / Correção Aplicada

**Quando usar:** quando o cliente pediu uma correção ou ajuste e o time já fez.

**Mensagem padrão:**
```
Oi [nome]! O ajuste que você pediu em [item] já foi feito.

[link atualizado, se aplicável]

Passa uma olhada e me confirma se ficou certo.
```

---

### 5.4 Resposta a Reclamação ou Insatisfação

**Quando usar:** quando o cliente trouxer uma crítica, erro ou insatisfação. Nunca deixar sem resposta no mesmo dia.

**Mensagem padrão:**
```
Oi [nome], obrigado por trazer isso.

Entendo o ponto e já vou levar para o time agora. Te retorno ainda hoje [ou: até [horário/data]] com um posicionamento.
```

**Após verificar internamente:**
```
[nome], levei para o time. [O que aconteceu em 1 linha]. [O que vai ser feito]. [Prazo].

Se quiser conversar sobre isso com mais calma, posso agendar um papo com o Danilo esta semana.
```

**Nota:** não dar satisfação antes de entender o que aconteceu. A primeira mensagem é só para mostrar que ouviu e vai apurar. A segunda é com a solução.

---

### 5.5 Boas Festas / Data Especial do Cliente

**Quando usar:** aniversário do cliente, datas comemorativas relevantes para o negócio dele.

**Mensagem padrão (aniversário do cliente):**
```
Oi [nome]! Feliz aniversário! Que seja um ótimo ano pra você.
```

**Nota:** curto e genuíno. Não transformar em mensagem de marketing.
