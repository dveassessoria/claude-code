Política de Privacidade

Aplicativo: DVE Ads  |  Última atualização: 24 de maio de 2026

Esta Política de Privacidade descreve como a DVE Assessoria coleta, usa, armazena e protege as informações acessadas por meio do aplicativo DVE Ads, integrado à plataforma Meta (Facebook/Instagram) via API de Marketing.

1. Quem somos

A DVE Assessoria é uma empresa brasileira especializada em RevOps (Revenue Operations), prestando serviços de gestão de marketing, tráfego pago e operações de vendas para seus clientes. O aplicativo DVE Ads é utilizado exclusivamente para gerenciar campanhas publicitárias dentro da plataforma Meta em nome dos clientes da DVE Assessoria.

2. Quais dados acessamos

Por meio da API de Marketing da Meta, o DVE Ads pode acessar:

  • Dados de campanhas, conjuntos de anúncios e anúncios das contas de anúncios autorizadas
  • Métricas de desempenho e relatórios de resultados (impressões, cliques, conversões, gastos)
  • Informações de configuração de públicos e segmentações
  • Criativos de anúncios (imagens, vídeos, textos) das contas autorizadas
  • Dados básicos do perfil da conta Meta associada (nome e ID da conta)

O aplicativo não coleta informações pessoais de usuários finais das campanhas, dados de cartão de crédito, senhas ou qualquer dado sensível que não esteja diretamente relacionado à gestão de campanhas publicitárias.

3. Como usamos os dados

Os dados acessados são utilizados exclusivamente para:

  • Criar, editar, pausar e otimizar campanhas publicitárias na Meta
  • Gerar relatórios de desempenho para os clientes da DVE Assessoria
  • Realizar análises internas para melhoria dos resultados de campanhas
  • Automatizar tarefas operacionais de gestão de tráfego pago

Os dados não são vendidos, compartilhados ou transferidos para terceiros que não sejam os próprios clientes titulares das contas.

4. Base legal (LGPD)

O tratamento de dados realizado pelo DVE Ads está fundamentado na Lei Geral de Proteção de Dados (Lei nº 13.709/2018), com base no legítimo interesse contratual entre a DVE Assessoria e seus clientes, e no consentimento expresso concedido pelo titular da conta ao autorizar o aplicativo dentro da plataforma Meta.

5. Armazenamento e segurança

Os dados acessados via API são processados em ambiente seguro e não são armazenados de forma permanente em servidores externos. Tokens de acesso são mantidos com criptografia e descartados quando o acesso é revogado ou encerrado.

Adotamos medidas técnicas e organizacionais adequadas para proteger as informações contra acesso não autorizado, perda acidental ou uso indevido.

6. Compartilhamento de dados

Os dados das contas de anúncios são acessados somente pela equipe da DVE Assessoria responsável pela gestão daquela conta específica. Nenhuma informação é compartilhada com outros clientes, parceiros ou terceiros sem autorização expressa do titular.

7. Retenção de dados

Os dados de acesso são mantidos apenas pelo período necessário à prestação do serviço contratado. Ao término do contrato ou mediante solicitação, o acesso é revogado e quaisquer dados locais são eliminados em até 30 dias.

8. Direitos do titular

O titular dos dados tem direito a:

  • Confirmar a existência de tratamento de seus dados
  • Acessar os dados tratados
  • Solicitar a correção de dados incompletos ou incorretos
  • Solicitar a exclusão dos dados tratados
  • Revogar o consentimento a qualquer momento, diretamente nas configurações do Meta

9. Exclusão de dados

Para solicitar a exclusão de dados ou revogar o acesso do aplicativo, o usuário pode:

  • Acessar as configurações de aplicativos na sua conta Meta e remover o DVE Ads
  • Entrar em contato diretamente pelo e-mail abaixo
